This is mainly just for me as a framework in hackathons to build vibecoded frontends and backends with partially hand-coded Python APIs. It is useful
because

Intial Setup: Create a new file ".env" with no name, only the file extension. Add your api keys as environment variable Use the .example if you need help

How to Run
Create your .env , and use the example if you need help

How to run:

```powershell
py -m pip install -r requirements.txt
py main.py
npm install express
node index.js
```

Open localhost:3000
